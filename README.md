# terio

**Интегратор интерфейсов.** Единое окно-терминал для работы с программами, LLM и скриптами.

terio выглядит как терминал: чёрный экран, ввод внизу, вывод наверх. Отличие — результат может быть любым окном: текст, видео, плеер, браузер. terio сам выбирает тип окна и передаёт ему фокус.

Под капотом: LLM для планирования, кеш скриптов для повторения успешных действий, песочница (CoW) для безопасного выполнения untrusted-команд, словарь синонимов для адаптации к манере пользователя.

## Быстрый старт

```bash
terio --help           # встроенная справка
terio run -- ls -la    # выполнить команду
terio mode quiet       # тихий режим (не отвлекает)
terio mode debug       # режим отладки (подтверждение каждого шага)
terio config show      # просмотр настроек
terio focus ↑          # переключить окно вывода
terio scroll 10        # скролл на 10 строк
terio repeat           # повторить последний запрос
```

## Ключевые возможности

- **Терминальная парадигма** — взаимодействие неотличимо от терминала
- **Rich-окна** — результат может быть плеером, браузером, графикой
- **Песочница (CoW)** — untrusted-команды в изолированном окружении
- **Скриптовая система** — всё управление через скрипты, пользователь может менять логику
- **Проактивный режим** — terio предугадывает следующий запрос
- **Экономия внимания** — по умолчанию тихий режим, без отвлечений
- **Словарь синонимов** — terio адаптируется к манере пользователя

## Команды

```bash
terio --help
terio run -- <command...>
terio mode [quiet|normal|debug]
terio config [show|set <key> <value>]
terio focus [↑|↓]
terio scroll <lines>
terio repeat
terio cost
```
