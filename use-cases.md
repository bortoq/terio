# Use Cases

Все сценарии terio работают по одному принципу: пользователь управляет программами через единую панель — terio. terio исполняет команды, рендерит результат и запоминает повторяющиеся последовательности.

## FLAC/CUE Album Split

Пользователь вводит `terio ask "split this flac/cue"`. Агент находит FLAC и CUE файлы, генерирует ffmpeg-команду, показывает трейс, исполняет. Результат — таблица треков. После 3 выполнений — готовый рецепт, выполняемый без LLM.

**Панель управления:** `ffmpeg`.

## Missing Episode In A Season

Пользователь: `terio ask "what's missing from season 3?"`. Агент инспектирует файлы через `ls`/`find`, сравнивает с ожидаемой нумерацией, показывает таблицу: какие есть, каких нет. Затем пользователь просит загрузить: `terio ask "download episode 3x07"`. Агент генерирует curl/yt-dlp/aria2c команду.

**Панель управления:** файловая система + download-инструменты.

## Resume Last Playlist

`terio ask "play my last playlist"`. Агент читает конфиг mpv, запускает `mpv --playlist=...`. terio показывает текущий трек как now-playing card с управлением (next/pause — hotkeys, которые передаются mpv).

**Панель управления:** `mpv`.

## File Copy With Progress

`terio run -- rsync -av --progress /src /dst`. terio парсит rsync output, показывает прогресс-бар, а после завершения — карточку: скорость, файлы, предупреждения.

**Панель управления:** `rsync`.

## GitHub Work Session

`terio ask "what needs attention in bortoq/terio?"`. Агент вызывает `gh issue list`, `gh pr list`, `git status`. Результат — приоритизированная лента. Пользователь может просмотреть PR, запустить тесты, запуштить — всё через shell-команды, без выхода из terio.

**Панель управления:** `gh` + `git`.

## Browser-Like News Reading

`terio ask "latest AI news"`. Агент запускает `curl` к RSS/news API, получает контент, форматирует как читаемую страницу. terio показывает лонгрид.

**Панель управления:** `curl` + AI-форматирование.

## Undo/Redo Safety

`terio run -- rm -rf ./old_project`. terio не выполняет `rm` напрямую — он перемещает файлы в `~/.terio/trash/`. `terio undo` восстанавливает их. Пользователь может безопасно экспериментировать.

**Панель управления:** файловая система (с защитой).

## Cost-Saving Routine

Пользователь регулярно просит одно и то же через агента. terio замечает повторения, компилирует рецепт. В следующий раз: «Выполнено по рецепту. Сэкономлено X токенов и Y секунд». Раздельно показана экономия внимания (нет переключений) и экономия LLM (нет токенов).

## Ключевой принцип

terio — **панель управления**. Он не становится браузером, плеером или Git-клиентом. Он даёт единый интерфейс для управления ими. Любую программу с CLI или API можно подключить к terio — через агента, рецепт или прямую shell-команду.
